🔒 How does this work?

This tool generates cryptographic hash digests using industry-standard algorithms. Hashes are one-way functions — you cannot reverse a hash to recover the original text.

MD5: 128-bit (32 hex chars). Deprecated for security use.
SHA-1: 160-bit (40 hex chars). Deprecated for security use.
SHA-256: 256-bit (64 hex chars). Widely used for security.
SHA-512: 512-bit (128 hex chars). Highest security level here.
